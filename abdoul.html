Vulnérabilité dans React Server Components

CVE-2025-55182 est une vulnérabilité critique de type « exécution de code à distance » (RCE) dans le protocole Flight de React. La chaîne d'attaque combine un traversement de chemin, l'injection d'un faux chunk et l'exploitation abusive du gestionnaire $B pour exécuter la fonction Function(attacker_code).

Thème :CVE-2025-55182

Date :2025

Auteur / Source :https://github.com/ejpir/CVE-2025-55182-research

Description :

Présentation de l'attaque L'exploit utilise trois champs de formulaire pour construire une charge utile malveillante : Crée un faux objet « chunk » avec une référence autoréférentielle (champ 1 $@0 → champ 0) Intègre une fausse réponse (« _response ») avec « _formData.get » défini sur « $1:constructor:constructor » Déclenche le gestionnaire « $B » qui appelle « response._formData.get(response._prefix + id) » Le traversement de chemin résout « _formData.get » → « Function », exécutant ainsi « Function(code) »

Versions concernées react-server-dom-webpack : 19.0.0, 19.1.0, 19.1.1, 19.2.0 react-server-dom-turbopack : mêmes versions Next.js : 15.x, 16.x (avant les correctifs), versions « canary » à partir de 14.3.0-canary.77+

Fixed Versions React: 19.0.1+, 19.1.2+, 19.2.1+ Next.js: 15.0.5, 15.1.9, 15.2.6, 15.3.6, 15.4.8, 15.5.7, 16.0.7+

Analyse :

Clôture de l'alerte. Cela ne signifie pas la fin d'une menace. Seule l'application de la mise à jour permet de vous prémunir contre l'exploitation de la vulnérabilité correspondante.
