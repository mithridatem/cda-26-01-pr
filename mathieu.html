MithridateM Router

Librairie PHP Composer pour simplifier le système de routing

Thème : Librairie PHP

Date : 26/06/2026

Auteur / Source : Mithridatem

Description :

Mithridatem Router est une bibliotheque PHP oriente PSR-4 qui fournit un mecanisme de routage HTTP minimaliste, testable et independant de tout framework.

1 API fluide pour declarer des routes GET, POST, Multi-Methode avec segments dynamiques {id} et wildcards /*.

2 Resolution de controlleurs parametres via references

3 Controle d'acces branchable grace a l'interface GrantCheckerInterface

4 Contexte de requete abstrait (RequestContextInterface)

5 Base path configurable pour les applications hebergees derriere un prefixe.

6 Exceptions explicites pour une gestion d'erreur claire.

Analyse :

Librairie simple à utiliser et intégrer

Avec la commande : composer require mithridatem/router
